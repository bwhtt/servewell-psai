# ServeWell - Public Servant Appreciation Index

Measuring how well U.S. states support and value their public workforce.

## About

The Public Servant Appreciation Index (PSAI) is a comprehensive framework that evaluates states across five dimensions:
- Competitive Compensation (30%)
- Workforce Health (28%)
- Public Perception & Trust (22%)
- Investment in People (15%)
- Recognition Infrastructure (5%)

## Project Structure

- `/website/` - Lovable-generated website code
- `/data/standardized/` - CSV data files powering the interactive map and reports
- `/content/` - Markdown content for all pages and state reports
- `/docs/` - Technical documentation and methodology
- `/prompts/` - AI prompts used in analysis
- `/templates/` - Reusable templates
